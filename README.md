AutoService

Консольное приложение на C# — учебный проект для закрепления принципов ООП.

Описание

Программа имитирует работу автосервиса:

очередь автомобилей на ремонт;
склад деталей с ограниченным количеством запчастей;
проверка неисправных деталей автомобиля;
пошаговый процесс ремонта;
расчёт оплаты и штрафов;
управление балансом автосервиса.
Реализовано
классы и объекты;
инкапсуляция;
композиция объектов;
работа с коллекциями (List, Queue);
конструкторы;
свойства;
перечисления (enum);
фабрика для создания объектов;
клонирование объектов;
обработка пользовательского ввода;
разделение ответственности между классами.
